# 📊 Business Process Support – Sales Data Analysis & Forecasting (Power BI)

This project focuses on analyzing sales data from selected products and applying forecasting techniques to support business decision-making using Power BI.

---

## 📌 Project Description

In this project, sales data for selected products was collected, cleaned, and analyzed. The goal was to extract meaningful insights and predict future trends.

Additionally, forecasting techniques were applied on a real-world dataset related to Orange to estimate future performance.

The results were visualized using interactive Power BI dashboards and presented in a professional presentation.

---

## 🛠️ Tools & Technologies

* Microsoft Excel (Data Cleaning & Preparation)
* Power BI (Data Visualization & Forecasting)
* Data Analysis Techniques

---

## 📊 Project Steps

1. 📦 Selected products for analysis
2. 📥 Collected sales-related data
3. 🧹 Cleaned and organized the data
4. 📈 Analyzed key metrics (sales, trends, performance)
5. 📊 Built interactive dashboards using Power BI
6. 🔮 Applied forecasting on Orange-related dataset
7. 🎤 Presented the results in a professional presentation
